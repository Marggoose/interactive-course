Цель модуля

В этом модуле студенты разработают и защитят свой собственный проект на основе знаний, полученных на предыдущих занятиях. Проект может быть как индивидуальным, так и командным. Важно продемонстрировать творческий подход, технические навыки и способность эффективно работать с технологией робототехники.

Содержание модуля

  • Занятие 4.1: Тематика проекта
  • Занятие 4.2: Разработка концепции и планирование проекта
  • Занятие 4.3: Сборка робота
  • Занятие 4.4: Тестирование и отладка робота

Тематика проекта

Вам предстоит выбрать одну из предложенных тем для вашего проекта. Вот несколько идей для вдохновения:

  • Соревновательный робот, который может маневрировать и избегать препятствий;
  • Робот, который выполняет определённую задачу (например, доставку предметов или уборку);
  • Робот с расширенной функциональностью, использующий датчики (например, для измерения расстояния или распознавания объектов);
  • Робот, управляемый через мобильное приложение или с использованием голосовых команд.

Этапы разработки проекта

Для успешной защиты проекта важно следовать определённой последовательности шагов:

  1. Выбор темы и обсуждение с преподавателем;
  2. Разработка концепции и планирование конструктивных решений;
  3. Проектирование и сборка робота;
  4. Программирование робота для выполнения заданных действий;
  5. Тестирование и отладка системы;
  6. Подготовка презентации и демонстрация проекта.

Форма контроля

Защита проекта: Студенты должны подготовить презентацию, которая включает описание проекта, демонстрацию работоспособности робота и ответ на вопросы преподавателя. Преподаватель будет оценивать оригинальность идеи, технические решения, а также способность студента представить свой проект и ответить на вопросы.
