О чём это занятие?

В этом уроке мы познакомимся с одной из самых популярных сред для обучения робототехнике — mBlock. mBlock использует блоковое программирование, что позволяет новичкам легко создавать программы, используя визуальные блоки вместо текста.

Мы начнём с установки mBlock, изучим его интерфейс и основные инструменты.

Шаг 1: Установка mBlock

Для начала скачайте и установите mBlock. Перейдите на официальный сайт mBlock и выберите версию для вашей операционной системы.

  1. Скачайте установочный файл для Windows или Mac.
  2. Запустите установку и следуйте инструкциям на экране.
  3. После установки запустите mBlock.

Шаг 2: Изучаем интерфейс mBlock

После установки мы откроем mBlock и изучим его интерфейс. Вот основные элементы, с которыми мы будем работать:

  • Область для блоков: здесь будут располагаться все доступные блоки программирования.
  • Область кода: место, где собираются блоки для создания программы.
  • Рабочая область: отображает ваш робот и его действия.

Шаг 3: Создаём первую программу

Теперь давайте создадим простую программу для управления светодиодом на роботе:

  1. В разделе «Блоки» выберите команду, которая включает светодиод.
  2. Перетащите этот блок в рабочую область.
  3. Настройте параметры блока, чтобы он включал светодиод на определённом пине робота.
  4. Нажмите кнопку «Запуск», чтобы увидеть результат.
