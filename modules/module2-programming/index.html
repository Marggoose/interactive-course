О модуле

В этом модуле мы познакомимся с двумя основными средами программирования, используемыми для создания программного обеспечения для роботов: mBlock и ArduinoIDE.

Мы рассмотрим, как они работают, их возможности, а также как настроить и начать использовать эти среды для программирования вашего робота.

Среда программирования mBlock

mBlock — это визуальная среда программирования, которая позволяет создавать программы с помощью блоков, перетаскиваемых мышью. Она идеально подходит для начинающих и детей, обучающихся робототехнике.

  • Простой и интуитивно понятный интерфейс
  • Визуальное программирование для роботов и других устройств
  • Поддержка различных платформ, включая Arduino

Среда программирования ArduinoIDE

ArduinoIDE — это среда для программирования микроконтроллеров Arduino. В отличие от mBlock, ArduinoIDE использует текстовое программирование с помощью языка C/C++. Это подходит для более сложных проектов и более опытных пользователей.

  • Поддержка разнообразных микроконтроллеров Arduino
  • Использует текстовый код для создания программ
  • Более гибкие настройки и возможности для сложных проектов

Что будем изучать в этом разделе?

В рамках этого модуля мы изучим:

  • Установку и настройку mBlock и ArduinoIDE
  • Основы программирования в обеих средах
  • Создание простых программ для управления роботами
