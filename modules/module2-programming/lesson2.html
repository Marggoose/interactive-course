О чём это занятие?

На этом занятии мы познакомимся с текстовой средой программирования ArduinoIDE, которая широко используется в проектах робототехники. Мы научимся устанавливать ArduinoIDE, подключать плату и загружать первую тестовую программу.

Шаг 1: Установка ArduinoIDE

Скачайте ArduinoIDE с официального сайта arduino.cc.

  1. Выберите вашу операционную систему и загрузите установочный файл.
  2. Установите программу, следуя стандартным инструкциям.
  3. Запустите ArduinoIDE.

Шаг 2: Интерфейс ArduinoIDE

Основные элементы интерфейса ArduinoIDE:

  • Редактор кода — здесь пишется программа на языке C/C++.
  • Кнопки компиляции и загрузки — для проверки и отправки кода на плату.
  • Монитор порта — для связи с микроконтроллером через COM-порт.

Шаг 3: Первая программа (Blink)

В ArduinoIDE по умолчанию есть пример «Blink» — мигание встроенным светодиодом. Он позволяет проверить работу платы и соединения.


// Программа мигает встроенным светодиодом на пине 13

void setup() {
  pinMode(13, OUTPUT); // Настройка пина 13 как выход
}

void loop() {
  digitalWrite(13, HIGH); // Включить светодиод
  delay(1000);            // Подождать 1 секунду
  digitalWrite(13, LOW);  // Выключить светодиод
  delay(1000);            // Подождать 1 секунду
}

Нажмите кнопку «Загрузить», чтобы отправить программу на плату и увидеть результат.

Форма контроля

Наблюдение: преподаватель проверяет, запустили ли учащиеся среду, собрали схему и успешно загрузили тестовую программу на плату.
