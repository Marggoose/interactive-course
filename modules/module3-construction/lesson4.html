Цель занятия

На этом занятии мы исследуем принцип работы моторов и датчиков для перемещения робота в пространстве. Учащиеся научатся программировать робота для выполнения различных манёвров, таких как повороты, движения по прямой, избегание препятствий.

Теория перемещения

Для перемещения робота используются моторы, которые управляются с помощью программируемой платы. Важно учитывать следующие моменты:

  • Скорость и направление вращения моторов;
  • Угол поворота для изменения направления;
  • Использование датчиков для обнаружения препятствий и корректировки траектории.

Практическое задание

Ваше задание — запрограммировать робота на выполнение следующих действий:

  1. Перемещение по прямой линии (на заданное расстояние);
  2. Поворот на 90 градусов влево или вправо;
  3. Движение назад на несколько сантиметров;
  4. Использование датчика для обхода препятствий (в случае их наличия на пути).

Для выполнения задания используйте среду программирования, с которой вы уже знакомы (mBlock или Arduino IDE).

Форма контроля

Наблюдение: преподаватель следит за корректностью выполнения манёвров, помогает при настройке программного обеспечения, оценивает точность выполнения заданий.
